# Elektrybałt

"Klapaucjusz uczynił to. Elektrybałt zadrżał od fundamentów do szczytu i zaczął:

Żądny młęciny brądnej, łydasty łaniele,

Samoćpaku mimajki...

Stanisław Lem, *Cyberiada*
